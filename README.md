# Intelligent Traffic Control System (Microservices + RL)

A hybrid AI system that optimizes traffic flow using a Java Spring Boot simulation environment and a Python Reinforcement Learning agent.

** Architecture**

Simulation Engine: Java Spring Boot (REST API)

AI Agent: Python (Q-Learning)

Visualization: Pygame (Real-time Digital Twin)

** How to Run**

Start the Java Server: ./mvnw spring-boot:run

Start the AI Agent: python traffic_viz.py
